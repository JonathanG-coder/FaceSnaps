# SnapFace 📸

## 📝 Description

**SnapFace** est une application web de partage de photos développée avec **Angular**. Ce projet a pour objectif de fournir une interface simple et conviviale permettant aux utilisateurs de :

- Partager leurs photos.
- "Snaper" (liker) les photos des autres utilisateurs.
- Découvrir et interagir avec les publications de la communauté.

## 🛠️ Technologies utilisées

- **Framework :** Angular
- **Langage :** TypeScript
- **Gestionnaire de paquets :** NPM
- **Stylisation :** SCSS
